Éléments HTML intégrés

HTML définit une longue liste de balises en ligne disponibles, dont une liste complète peut être trouvée sur le Mozilla Developer Network .

  • Pour mettre du texte en gras , utilisez <strong>.
  • Pour mettre du texte en italique , utilisez <em>.
  • Les abréviations, telles que HTML, doivent utiliser <abbr>, avec un titleattribut facultatif pour la phrase complète.
  • Les citations, comme — Mark Otto , devraient utiliser <cite>.
  • Suppriméle texte doit utiliser <del>etinséréle texte doit utiliser <ins>.
  • Le texte en exposant utilise <sup>et le texte en indice utilise <sub>.

La plupart de ces éléments sont stylés par les navigateurs avec peu de modifications de notre part.
